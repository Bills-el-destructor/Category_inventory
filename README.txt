It is a small project where you can create categories and list them.
Every time you create a new category, it will show you on a blank page with the creation date and the image.
Both form fields have validations with rules and validations
This Request is called StoreCategory

This project uses a Docker library. This is called Vessel and you can see its documentation and installation
on its official website that is this: https://vessel.shippingdocker.com/
